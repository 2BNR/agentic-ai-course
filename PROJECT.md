# Agentic AI Course 2026

## What this is
A self-paced course website for learning Agentic AI, built section by section. 16 sections total.

## Structure
- index.html — course homepage
- style.css — shared styles (dark theme, Syne + JetBrains Mono fonts)
- course.js — progress tracking via localStorage
- sections/01-intro.html — fully built example (use as template)
- sections/02 through 16 — placeholders to be filled in

## How to build a section
Each section needs:
1. Theory: written explanations + visual HTML/CSS diagrams
2. Concept cards grid
3. Comparison tables where relevant
4. Callout boxes (info, success, warning)
5. Step-by-step lab with syntax-highlighted code blocks
6. Study with Claude panel with 3-4 prompt chips
7. Mark complete button + next/prev navigation

## Design language
Dark bg (#060609), accent purple (#6c63ff), green (#00e5a0)
Fonts: Syne (headings), DM Sans (body), JetBrains Mono (code/labels)
See style.css for all CSS variables and component classes.
