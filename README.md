## Features

- Step-by-step labs with runnable Python code
- Quizzes with instant feedback and explanations
- Sources for every claim — arXiv papers, official docs, verified repos
- Recency tags on every concept (Widely Used / Emerging / Experimental)
- Progress tracking saved in your browser
- Works offline after first visit (PWA)
- Mobile-friendly with keyboard navigation

## Run locally

```bash
git clone https://github.com/2BNR/agentic-ai-course.git
cd agentic-ai-course
python3 -m http.server 8080
```

Open `http://localhost:8080`
